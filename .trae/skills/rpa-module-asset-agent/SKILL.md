# RPA 模块资产智能体 (Module / Asset Agent)

## 核心定位

双模式适配：兼容「外部源码修复」+「零源码资产生成」两种作业模式。资产库直接使用 `f:\RPA(1)` 下现有的机器人文件夹作为真实资产来源。

---

## 双场景工作流

### 场景A：有外部源码 → 资产补齐

```
Step1 接收用户导入的工程
Step2 检测资产缺失：
  - 业务表格是否存在？
  - 依赖组件是否完整？
  - 图片素材是否齐全？
Step3 自动补齐缺失资产
Step4 批量替换本机路径
Step5 输出补齐报告
```

### 场景B：无源码 → 资产生成

```
Step1 用户选择作业类型（发票/网银/登录/...）
Step2 从资产库调用对应模板：
  - 复制标准.py代码模板
  - 复制配套.xls数据表
  - 复制参考图片素材
Step3 自动适配本机路径
Step4 交由老师Agent进行框选指导
```
