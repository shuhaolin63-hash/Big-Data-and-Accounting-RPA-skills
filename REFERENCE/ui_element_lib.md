# UI 元素选择器参考库

本文档按机器人分类列出各 UI 元素的 selector 定位方式，供框选和代码编写参考。选择器路径为示例值，实际使用时需要根据本机页面实际情况进行框选拾取。

## 一、电子税务局登录页

### 1.1 页面基本信息

| 项目 | 说明 |
|------|------|
| URL | https://etax.chinatax.gov.cn/ （示例） |
| 浏览器 | Chrome / Edge |
| 主要元素 | 用户名输入框、密码输入框、登录按钮、登录头像按钮 |

### 1.2 元素选择器

#### 用户名输入框

| 定位方式 | 选择器路径 |
|----------|-----------|
| CSS | `input#username` 或 `input[name="username"]` |
| XPath | `//input[@id="username"]` |
| 框选建议 | 选中 input 元素本身，不要包含 label 文本 |

#### 密码输入框

| 定位方式 | 选择器路径 |
|----------|-----------|
| CSS | `input#password` 或 `input[name="password"]` |
| XPath | `//input[@id="password"]` |
| 框选建议 | 选中 input[type="password"] 元素，注意区分密码框和普通输入框 |

#### 登录按钮

| 定位方式 | 选择器路径 |
|----------|-----------|
| CSS | `button#loginBtn` 或 `button[type="submit"]` |
| XPath | `//button[@id="loginBtn"]` |
| 框选建议 | 选中 button 元素，注意排除"重置"或"取消"按钮 |

#### 登录头像按钮（切换用户）

| 定位方式 | 选择器路径 |
|----------|-----------|
| CSS | `div.avatar-wrapper` 或 `img.avatar` |
| XPath | `//div[contains(@class,"avatar")]` |
| 框选建议 | 选中头像外层容器或 img 图片元素 |

## 七、Excel 表格操作

### 7.1 表格区域框选

| 操作 | 选择器 / 范围说明 |
|------|-------------------|
| 选择工作表 | `sheet = workbook.sheet_by_index(0)` 或 `sheet_by_name("Sheet1")` |
| 数据区域 | 从 A1（表头）到最后一行的最后一列，如 `A1:F10` |
| 表头行 | 第 1 行（索引 0），用于字段名识别 |
| 数据行 | 从第 2 行开始，到有数据的最后一行 |

### 7.2 框选精度要求

| 范围标准 | 说明 | 评分影响 |
|----------|------|----------|
| 精准范围 | 仅选择实际有数据的表格区域 | 满分 |
| 多选空行 | 包含了数据区域外的空白行 | 扣 5-10 分 |
| 多选空列 | 包含了数据区域外的空白列 | 扣 3-5 分 |
| 少选数据行 | 遗漏了部分数据行 | 扣 10-15 分 |
| 少选数据列 | 遗漏了必要字段列 | 扣 10-15 分 |

## 八、通用 UI 操作选择器参考

| 元素类型 | 通用 CSS 选择器模式 | 通用 XPath 模式 |
|----------|---------------------|-----------------|
| 文本输入框 | `input[type="text"]`、`input:not([type])` | `//input[@type="text"]` |
| 密码输入框 | `input[type="password"]` | `//input[@type="password"]` |
| 提交按钮 | `button[type="submit"]`、`input[type="submit"]` | `//button[@type="submit"]` |
| 链接 | `a[href]` | `//a[@href]` |
| 复选框 | `input[type="checkbox"]` | `//input[@type="checkbox"]` |
| 下拉框 | `select` | `//select` |
| 表格 | `table` | `//table` |
| 图片 | `img` | `//img` |
| 弹窗 | `div.modal`、`div.dialog` | `//div[contains(@class,"modal")]` |
| iframe | `iframe` | `//iframe` |

## 注意事项

1. **选择器验证**：所有选择器路径需要通过浏览器开发者工具（F12）验证可用性
2. **动态页面**：对于动态加载的页面元素，需要增加足够的等待时间
3. **页面更新**：当系统页面改版后，选择器路径需要重新拾取
4. **本机差异**：不同机器上的页面渲染可能略有差异，建议每台机器独立进行框选
