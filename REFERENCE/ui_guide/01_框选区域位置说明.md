# 框选区域位置说明

本文档详细说明 RPA 作业中各框选区域的页面位置、框选起始/结束范围、元素类型和功能用途。每一处框选都直接影响作业的运行效果和评分结果。

---

## 一、电子税务局登录页

### 1.1 用户名输入框

| 项目 | 说明 |
|------|------|
| **页面位置** | 电子税务局登录页 -> 登录表单区域 -> 用户名输入框（表单第一行） |
| **框选起始范围** | 从输入框左侧边缘（约距页面左边界 30%）到右侧边缘 |
| **框选结束范围** | 包含整个 input 元素，不含 label 外部容器 |
| **元素类型** | input[type="text"] / input[type="email"] |
| **功能用途** | 输入登录用户名（通常为纳税人识别号或手机号） |
| **参考图片** | `1641794104431.png`（登录界面）：表单上部第一个输入框 |
| **注意事项** | 部分版本为"用户名/手机号/身份证"三合一输入框，确认对应页面版本 |

### 1.2 密码输入框

| 项目 | 说明 |
|------|------|
| **页面位置** | 电子税务局登录页 -> 登录表单区域 -> 密码输入框（表单第二行） |
| **框选起始范围** | 从输入框左侧边缘到右侧边缘，紧贴 input 元素边框 |
| **框选结束范围** | 仅包含 input[type="password"] 元素 |
| **元素类型** | input[type="password"] |
| **功能用途** | 输入登录密码 |
| **参考图片** | `1641794104431.png`（登录界面）：表单上部第二个输入框 |
| **注意事项** | 密码框输入内容会自动隐藏为圆点，这是正常行为，不影响功能 |

### 1.3 登录按钮

| 项目 | 说明 |
|------|------|
| **页面位置** | 电子税务局登录页 -> 登录表单区域 -> 登录按钮（表单最下方） |
| **框选起始范围** | 从按钮左侧边缘到右侧边缘 |
| **框选结束范围** | 仅包含 button 元素，或者具有 "登录" 文本的 a/span 元素 |
| **元素类型** | button[type="submit"] / a.login-btn |
| **功能用途** | 点击后提交用户名和密码进行登录验证 |
| **参考图片** | `1641794104431.png`（登录界面）：表单底部蓝色/绿色按钮 |
| **注意事项** | 有些版本登录按钮在表单外，需要根据实际页面定位 |

### 1.4 登录头像按钮（切换用户）

| 项目 | 说明 |
|------|------|
| **页面位置** | 电子税务局登录页 -> 右上角 -> 用户头像图标 |
| **框选起始范围** | 从头像图片左上角到右下角 |
| **框选结束范围** | 包围头像图片及其外圈圆形边框 |
| **元素类型** | img / div.avatar |
| **功能用途** | 点击后弹出用户选择/切换菜单 |
| **参考图片** | `1641794104431.png`（登录界面）：页面右上角圆形头像 |
| **注意事项** | 若使用账号密码登录则无需框选此元素 |

## 二、Excel 数据表格区域

### 2.1 数据表头行

| 项目 | 说明 |
|------|------|
| **页面位置** | Excel 工作表 -> 第 1 行（或指定的表头行） |
| **框选起始范围** | 从第 1 列到最后一列（如 A1 到 F1） |
| **框选结束范围** | 包含所有表头单元格，整行选中 |
| **元素类型** | Excel 工作表行（row） |
| **功能用途** | 识别字段名称，用于数据映射和字段匹配 |
| **注意事项** | 表头行建议单独框选或作为数据区域的一部分包含进来 |

### 2.2 数据内容区域

| 项目 | 说明 |
|------|------|
| **页面位置** | Excel 工作表 -> 从表头下一行到数据最后一行 |
| **框选起始范围** | 从 A2（第 2 行第 1 列）开始 |
| **框选结束范围** | 到有数据的最后一行最后一列（如 F10） |
| **元素类型** | Excel 工作表区域（range） |
| **功能用途** | 读取业务数据用于循环处理 |
| **框选精度要求** | 精准包含所有数据行，不包含多余空白行 |
| **注意事项** | 如果数据行数不固定，建议使用动态范围检测 |

## 四、发票开具系统页面

### 4.1 发票开具表单区域

| 项目 | 说明 |
|------|------|
| **页面位置** | 电子税务局 -> 发票开具模块 -> 发票开具表单 |
| **框选起始范围** | 从表单左上角开始，覆盖所有需要填写的字段 |
| **框选结束范围** | 到表单右下角，包括所有输入框和下拉框 |
| **元素类型** | form / div.invoice-form |
| **功能用途** | 填写发票开具所需的购买方信息和商品信息 |

| **注意事项** | 表单可能包含多个子区域（购买方、商品、备注等） |

## 五、退出/注销按钮

| 项目 | 说明 |
|------|------|
| **页面位置** | 系统页面右上角 -> 用户头像/用户名旁边 -> "退出"/"注销" |
| **框选起始范围** | 从按钮/链接文本左侧到右侧 |
| **框选结束范围** | 选中整个 a / button / span 元素 |
| **元素类型** | a.logout / button.logout-btn |
| **功能用途** | 安全退出当前登录会话 |
| **注意事项** | 注意区分"退出"和"返回"按钮，退出是注销当前用户 |
