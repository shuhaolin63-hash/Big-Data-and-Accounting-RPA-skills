# 框选操作记录模板

本文档用于记录 RPA 作业中每一步框选操作的详细信息。每完成一处框选，请立即填写对应行，确保记录完整。

---

## 填写说明

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| 时间戳 | 框选操作的时间 | 2026-07-19 10:30:00 |
| 框选对象 | 被框选的元素名称 | 用户名输入框 |
| 框选位置 | 页面位置描述 | 电子税务局登录页 -> 登录表单 -> 用户名输入框 |
| 框选范围 | 框选的坐标/选择器 | CSS: input#username 或 XPath: //input[@id="username"] |
| 对应功能 | 该框选实现的 RPA 功能 | 文本输入（type）用于输入用户名 |
| 操作状态 | 框选是否成功 | 成功 / 失败 / 需调整 |
| 备注 | 其他说明 | 框选范围偏移约 3px，需微调 |

### 状态判定规则

| 状态 | 判定标准 |
|------|----------|
| 成功 | 框选精准选中目标元素，偏移 < 5px，功能正常 |
| 需调整 | 框选存在轻微偏差，偏移 5-15px，功能偶发异常 |
| 失败 | 框选严重偏离目标或选错元素，功能无法执行 |
