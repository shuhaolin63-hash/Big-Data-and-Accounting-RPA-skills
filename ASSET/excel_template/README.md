## 资产库使用说明

1. **模板复制**：`asset_auto_copy.py` 根据场景类型（`invoice_check` / `invoice_open` / `invoice_certify` / `bank_pay` / `bank_receive` / `bank_corp_pay` / `stock`）自动复制对应表格到用户工作区
2. **格式说明**：所有表格均为 `.xls` 格式（兼容 Excel 97-2003），使用 `xlrd` 库读取
3. **路径引用**：表格原始路径为 `f:\RPA(1)\{文件名}`，复制到工作区后由 `path_batch_replace.py` 自动适配本机路径
4. **数据保护**：表格为只读模板，不应直接修改原始文件
